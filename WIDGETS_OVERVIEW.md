## 🚀 Usage Examples

### Example 1: Complete Payment Flow
```
User: "Create a new account"
→ generate-fufa-account widget renders
→ User fills form and submits
→ Account created with FUFA handle

User: "Sign me in"
→ sign-in widget renders
→ User enters credentials
→ Logged in, balance displayed

User: "Check my balance"
→ check-balance widget renders
→ Shows ₹5,000.00 available

User: "Send ₹500 to jane.smith@fufa"
→ send-money widget renders
→ Scam analysis runs
→ Transaction completes
```

### Example 2: Security Flow
```
User: "Send ₹10,000 to unknown.handle@fufa"
→ send-money widget renders
→ Scam analysis detects HIGH risk
→ Security alert displayed
→ User must confirm to proceed
→ Transaction blocked or approved

User: "Analyze this transaction for scam"
→ analyze-scam-risk widget renders
→ Shows risk score: 92/100
→ Lists detected threats
→ Provides recommendations
```

### Example 3: Financial Products
```
User: "Check my pension status"
→ pension-status widget renders
→ Shows 2 active enrollments
→ Displays payout history
→ Shows PRAN numbers and amounts

User: "Check my insurance"
→ insurance-status widget renders
→ Shows 2 active policies
→ Displays coverage amounts
→ Shows claim history
```
